====================
BROWSER

Legyen két cache:
  DATA-OVER
  DATA-MAIN
  DATA-UNDER

A méretük legyen 50 000 sor.
Az átfedés legyen közöttük a tényleges képernyősorszám kétszerese.
Képernyősorszám jelenleg 74 teljes képernyőn.

Kezdetben a MAIN 0-tól kezdődik, az OVER üres, a SHADOW pedig (- 50001 74 74) = 49853-től.

Felfelé görgetéskor:
  ha a pozíció MAIN-ben az átfedésen belül van és a MAIN kezdete nagyobb mint alsó határ (0):
    az OVER-ből MAIN lesz
	a MAIN-ből UNDER lesz
	az UNDER-ből OVER lesz és fel kell tölteni lekérdezéssel

Lefelé gorgetéskor:
  ha a pozíció a MAIN alsó átfedésében van és a MAIN vége kisebb mint a felső határ (?)
    az UNDER-ből MAIN lesz
	a MAIN-ből OVER lesz
	az OVER-ből UNDER lesz, és fel kell tölteni lekérdezéssel

INICIALIZÁLÁS

DATA-COUNT = count                                                       ; jelen lekérdzés hány soros?
DATA-MAIN  = select FIRST-ROW offsettől (min DATA-LIMIT DATA-COUNT)      ; lekérdezünk a fő bufferbe a buffer limit vagy a lekérdezés méretéig (amelyik kisebb)
DATA-UNDER = ha DATA-COUNT > DATA-LIMIT: select (- DATA-LIMIT (* ROW-COUNT 2)) offsettől (mint DATA-LIMIT DATA-COUNT)-ig
DATA-OVER  = nil                                                         ; az elején vagyunk, nincs fölötte semmi
SELECTION  = '()

Select id into new table. Use that to maintain query consistency while browsing results.
New table should not be a temp table, as temp tables are dropped as soon as connection is closed.

; Selecting IGENYLES_ID into new table
create table temp0001 as
  select igenyles_id from igenylesek
    left join szerv_egysegek on igenylesek.szerv_egys_id = szerv_egysegek.szerv_egys_id
	left join tank_kozpontok on szerv_egysegek.tank_kozpont_id = tank_kozpontok.tank_kozpont_id
	where tank_kozpontok.tank_kozpont = "Egri Tankerületi Központ";

; Selecting columns from db, rows according to temp table
select IGENYLES_ID, TANK_KOZPONT, SZERV_EGYS, BEOSZTAS, VISELT_CSALADNEV, VISELT_UTONEV_1, VISELT_UTONEV_2, SZUL_CSALADNEV, SZUL_UTONEV_1, SZUL_UTONEV_2, ANYA_CSALADNEV, ANYA_UTONEV_1, ANYA_UTONEV_2, ORSZAG, VAROS, SZUL_DATUMA, IGAZOLVANYSZAM, TELEFON, EMAIL, T_SOROZATSZAM, T_ERVENYESSEG_KEZDETE, T_ERVENYESSEG_VEGE, T_TIPUS, T_KAPCS_ESZKOZ, T_ALLAPOT, T_VISSZAVONAS_DATUMA
  FROM (IGENYLESEK)
  LEFT JOIN SZERV_EGYSEGEK ON IGENYLESEK.SZERV_EGYS_ID = SZERV_EGYSEGEK.SZERV_EGYS_ID
  LEFT JOIN BEOSZTASOK ON IGENYLESEK.BEOSZTAS_ID = BEOSZTASOK.BEOSZTAS_ID
  LEFT JOIN SZEMELYEK ON IGENYLESEK.SZEMELY_ID = SZEMELYEK.SZEMELY_ID
  LEFT JOIN T_TIPUSOK ON IGENYLESEK.T_TIPUS_ID = T_TIPUSOK.T_TIPUS_ID
  LEFT JOIN T_KAPCS_ESZKOZOK ON IGENYLESEK.T_KAPCS_ESZKOZ_ID = T_KAPCS_ESZKOZOK.T_KAPCS_ESZKOZ_ID
  LEFT JOIN T_ALLAPOTOK ON IGENYLESEK.T_ALLAPOT_ID = T_ALLAPOTOK.T_ALLAPOT_ID
  LEFT JOIN TANK_KOZPONTOK ON SZERV_EGYSEGEK.TANK_KOZPONT_ID = TANK_KOZPONTOK.TANK_KOZPONT_ID
  LEFT JOIN VAROSOK ON SZEMELYEK.VAROS_ID = VAROSOK.VAROS_ID
  LEFT JOIN ORSZAGOK ON VAROSOK.ORSZAG_ID = ORSZAGOK.ORSZAG_ID
  where igenylesek.igenyles_id in (select IGENYLES_ID from temp0001);
